# My Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, and professional experience as a Full Stack Developer.

## 🚀 Overview

I built this portfolio using Next.js and TypeScript to present my work in an interactive, visually appealing format. The site features dynamic horizontal scrolling sections, smooth animations, and a professional contact system with email functionality.

## ✨ Features

- **Dynamic Sections**: Interactive horizontal-scrolling portfolio with About, Skills, Projects, Experience, Resume, and Contact sections
- **Responsive Design**: Optimized for all device sizes with mobile-first approach
- **Email Integration**: Direct contact form with Gmail integration using Nodemailer
- **File Converter**: Built-in PDF to text converter for resume processing
- **Animated UI**: Smooth transitions and micro-interactions throughout
- **Professional Templates**: Pre-built email templates for quick communication

## 🛠 Tech Stack

- **Framework**: Next.js 15.3.3 with TypeScript
- **Styling**: Tailwind CSS with custom animations
- **UI Components**: Shadcn/ui component library
- **Icons**: Lucide React icons
- **Email**: Nodemailer with Gmail SMTP
- **Deployment**: Vercel (or your preferred platform)

## 📂 Project Structure

```
src/
├── app/                 # Next.js app router pages
├── components/
│   ├── horifolio/      # Main portfolio section components
│   └── ui/             # Reusable UI components
├── hooks/              # Custom React hooks
└── lib/                # Utility functions
```

## 🎯 Key Sections

- **About**: Personal introduction and core skills
- **Skills**: Categorized technical and soft skills with animated separators
- **Projects**: Showcase of development projects with GitHub and live demo links
- **Experience**: Professional work history with detailed achievements
- **Resume**: Interactive resume viewer with file conversion capabilities
- **Contact**: Professional contact form with email templates

## 📧 Contact Integration

I've implemented a direct email system that allows visitors to:
- Send messages using pre-built professional templates
- Contact me without email client redirects
- Receive immediate confirmation of message delivery

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/eyesee11/portfolio.git

# Install dependencies
npm install

# Set up environment variables
cp .env.local
# Add your Gmail credentials for email functionality

# Run development server
npm run dev
```
