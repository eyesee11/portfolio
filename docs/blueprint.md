# **App Name**: HoriFolio

## Core Features:

- Horizontal Section Display: Display portfolio sections horizontally, creating a non-traditional browsing experience.
- Interactive Section Expansion: Widen the active section horizontally within the same space when tapped, providing a more immersive view of its content while maintaining vertical dimensions.
- Mouse-activated Scroll: Enable smooth horizontal scrolling by hovering the mouse to the right side.
- Contact section: Include a section displaying contact information.

## Style Guidelines:

- Primary color: Pale sand (#D8C9AE) to evoke creativity and sophistication.
- Background color: Charcoal Gray (#575757) for a gentle backdrop.
- Body and headline font: 'Inter', sans-serif.
- Employ a horizontal layout with sections spanning the viewport width. Section width should increase when a section is tapped, maintaining vertical dimensions.
- Implement smooth, easing transitions for section expansion and mouse-activated scrolling.
